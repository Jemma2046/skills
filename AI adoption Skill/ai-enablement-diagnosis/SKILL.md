# AI Enablement 阻力诊断与干预

## 任务目标
- 本Skill用于：Enablement与员工1v1沟通时，快速、标准化诊断AI学习阻力类型
- 能力包含：10题问答收集、4类阻力自动诊断、个性化干预方案生成、沟通话术输出
- 触发条件：需要推动员工使用AI工具、了解员工抵触原因、准备1v1赋能沟通

## 操作步骤

### 第一步：收集员工基础信息
向员工确认以下信息：
- 员工岗位（如：代理人/内勤/主管）
- 入职年限（数字）

### 第二步：引导完成10题问答

**封闭式问题（快速标记）**：

**Q1 - 接触意愿（惯性）**
你对新AI工具的第一感受？
- A. 完全不想了解，习惯旧方法
- B. 好奇但观望，怕麻烦
- C. 愿意了解，想试试
- D. 非常期待，想马上用

**Q2 - 学习顾虑（惰性，可多选）**
你觉得用AI最大的顾虑是？
- A. 太复杂，学不会
- B. 没时间，耽误业绩
- C. 记不住，用不好
- D. 没顾虑，愿意学

**Q3 - 心理担忧（情绪，可多选）**
你担心AI会带来什么影响？
- A. 替代自己，丢工作
- B. 操作出错，影响客户
- C. 被公司监控、考核
- D. 没担心，觉得是帮手

**Q4 - 接受态度（抵触）**
如果公司要求你用AI，你的感受是？
- A. 反感，坚决不用
- B. 抵触，敷衍应付
- C. 中立，看效果再说
- D. 配合，愿意尝试

**Q5 - 工具认知（惯性）**
你认为AI工具对你的工作？
- A. 没用，不如人工
- B. 一般，可替代可不用
- C. 有用，能帮省时间
- D. 很有用，能提升业绩

**Q6 - 时间成本（惰性）**
你愿意花多久时间学习AI基础操作？
- A. 3分钟以内
- B. 10分钟以内
- C. 30分钟以内
- D. 不愿意花时间

**Q7 - 风险感知（情绪）**
你觉得试用AI的风险？
- A. 风险很大，怕出错
- B. 有一点风险，能接受
- C. 没风险，很安全
- D. 不清楚

**Q8 - 自主意愿（抵触）**
你使用AI的前提是？
- A. 自愿选择，没人强迫
- B. 公司推荐，不强求
- C. 领导要求，不得不做
- D. 无所谓，听安排

**开放式问题（补充细节）**：

**Q9 - 负面反馈**
你觉得AI工具最没用/最麻烦的地方是什么？
（记录员工原话，用于深度理解阻力来源）

**Q10 - 需求期望**
如果让你用AI，你希望它先帮你解决什么问题？
（记录员工期望，用于设计激励切入点）

### 第三步：运行诊断脚本

将收集的信息整理为JSON格式，运行诊断脚本：

```bash
python scripts/diagnosis.py --input '{
  "employee": {"position": "代理人", "years": 3},
  "closed_questions": {
    "Q1": "A", "Q2": ["A", "B"], "Q3": ["A"], "Q4": "A",
    "Q5": "A", "Q6": "D", "Q7": "A", "Q8": "C"
  },
  "open_questions": {
    "Q9": "习惯了老方法，觉得AI太复杂",
    "Q10": "希望先帮我处理重复性工作"
  }
}'
```

### 第四步：解读报告并执行干预

脚本输出包含：
1. **阻力类型与等级**：惯性/惰性/情绪/抵触及其严重程度
2. **诊断依据**：触发规则的具体问答结果
3. **干预方案**：针对各阻力类型的具体措施
4. **沟通话术**：可直接对员工说的2-3句话

优先处理阻力等级"高"的类型，选择性执行干预方案。

## 使用示例

### 示例1：高惯性+中情绪员工
- 场景/输入：
  - 岗位：内勤，入职5年
  - Q1=A, Q5=A, Q3含A, Q4=A/B
- 预期产出：惯性(高)+情绪(中)诊断报告
- 关键要点：优先消除情绪恐惧，再建立新习惯

### 示例2：纯惰性员工
- 场景/输入：
  - 岗位：代理人，入职2年
  - Q2含A/B/C, Q6=A/D, Q9提到"复杂/麻烦"
- 预期产出：惰性(高)诊断报告
- 关键要点：承诺零门槛、降低心理负担

### 示例3：多类型混合阻力
- 场景/输入：同时触发惯性+惰性+抵触
- 预期产出：综合诊断报告含多类型干预方案
- 关键要点：按优先级排序干预方案

## 注意事项

- 问答过程保持中立态度，不评判员工选择
- 开放式问题(Q9/Q10)记录原话，不加工直接传入脚本
- 报告仅作为1v1沟通参考，实际执行可根据情境调整
- 多类型阻力叠加时，优先处理情绪类（恐惧/担忧）
